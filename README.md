# Orion
Community driven platform for sharing, discussing and rating content.

## :small_red_triangle_down: Spec.
- Guest users may:
  - Browse communities
  - Open and read posts by members of a community
  - Read comments
- Signed in users may:
  - Join community
  - Create posts
  - Comment on posts

note: Plus different sorting options.
